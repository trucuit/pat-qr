# Pat-QR

Ứng dụng React Native quét mã QR, xây dựng với Ignite boilerplate — phiên bản PAT (Personal Access Token) cho quản lý QR.

## 📋 Tính năng

- Quét mã QR và Barcode
- Quản lý danh sách QR codes
- React Navigation
- MobX State Tree (MST)
- Hỗ trợ Android & iOS

## 🛠️ Công nghệ

- **Framework:** React Native (Expo)
- **Language:** TypeScript
- **Boilerplate:** [Ignite](https://github.com/infinitered/ignite)
- **State:** MobX State Tree

## ⚙️ Cài đặt

### Yêu cầu

- Node.js >= 18
- Yarn
- Expo CLI

### 1. Clone & cài đặt

```bash
git clone https://github.com/trucuit/pat-qr.git
cd pat-qr
yarn install
```

### 2. Chạy ứng dụng

```bash
npx expo start
```

### 3. Testing

```bash
yarn test                # Unit tests
yarn test:maestro        # E2E (Maestro)
```

## 🔒 Bảo mật

Không yêu cầu API key. Camera permission request lúc runtime.
